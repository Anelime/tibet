# Тур в Тибет: кора вокруг Кайласа — Trekki.travel

Одностраничный лендинг авторского тура в Тибет (21.09–04.10.2026): кора вокруг Кайласа, базовый лагерь Эвереста, Лхаса, Шигадзе, озеро Манасаровар. Программа по дням, гиды, цены и условия, отзывы и форма заявки.

## Структура

- `index.html` — лендинг (всё в одном файле: разметка, стили, скрипты)
- `assets/tibet/` — фотографии тура и гида
- `assets/trekki-logo*.png` — логотипы Trekki.travel
- `assets/favicons/` — иконки сайта, сгенерированные из логотипа

Шрифты подключаются через Google Fonts (Inter, Oswald), внешних сборок и зависимостей нет.

Форма заявки работает без бэкенда: кнопка «Отправить заявку» открывает WhatsApp гида с готовым текстом из заполненных полей.

## Локальный запуск

Открыть `index.html` в браузере или поднять статический сервер:

```bash
python3 -m http.server 8000
```

## Деплой

Сайт публикуется через GitHub Pages из ветки `main` (корень репозитория): https://anelime.github.io/tibet/
